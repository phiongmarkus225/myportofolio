# 📊 Portfolio Data Scientist

Selamat datang di portfolio profesional saya sebagai Data Scientist! Website ini menampilkan ringkasan perjalanan, keahlian, dan proyek-proyek terbaik saya secara interaktif dan modern.

## Tentang Portfolio
Portfolio ini dibuat menggunakan **Streamlit** agar mudah diakses, responsif di berbagai perangkat, dan menampilkan informasi secara visual serta interaktif. Anda dapat melihat:

- **Profil Singkat & Statistik**
- **Pengalaman & Pencapaian**
- **Daftar Proyek Pilihan**
- **Keahlian Teknis**
- **Riwayat Pendidikan & Sertifikasi**
- **Kontak & Social Media**

Semua konten difokuskan untuk memperkenalkan siapa saya, keahlian utama, dan hasil nyata yang pernah saya capai, tanpa membongkar detail kode program.
